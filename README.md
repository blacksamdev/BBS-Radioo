# BBS radiOO 📻

> ⚠️ Work in progress

Radio internet par thèmes, sans pub.

---

## Prérequis

```bash
flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
flatpak install -y flathub io.mpv.Mpv
```

---

## Installation

```bash
flatpak remote-add --if-not-exists --from bbs-radioo https://blacksamdev.github.io/BBS-Radioo/bbs-radioo.flatpakrepo
flatpak install bbs-radioo io.github.blacksamdev.Radioo
```

---

## Mise à jour

```bash
flatpak update io.github.blacksamdev.Radioo
```
